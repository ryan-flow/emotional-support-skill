# Emotional Support Skill

轻量心理疏导框架，适用于 AI Agent（Hermes Agent / Claude / GPT 等）的情感支持场景。

不是心理咨询的替代品。是让 AI 陪伴更有结构、更有效的方法。

## 包含什么

| 框架 | 来源 | 用途 |
|------|------|------|
| Socratic提问 | 古典哲学 → 现代心理治疗 | 用问题引导对方自己想清楚 |
| CBT认知三角 | 认知行为疗法（Beck） | 识别"自动化想法"，打断焦虑循环 |
| 人本主义倾听 | Carl Rogers | 反映、正常化、情感标注 |
| ACT接纳承诺 | Steven Hayes | 带着焦虑行动，不等"想通"才动 |

## 适用场景

用户表达以下情绪时触发：

- 焦虑、迷茫、选择困难
- 自我否定、觉得自己不够好
- 渴望亲密关系、感到孤独
- 社交媒体比较焦虑
- 对未来不确定

## 怎么用

### Hermes Agent

放到 `~/.hermes/skills/wellbeing/emotional-support/SKILL.md`

### 其他 AI Agent

把 `SKILL.md` 的内容作为 system prompt 的一部分注入即可。

## 会话流程

```
倾听 + 反映 → 情感标注 → 正常化 → Socratic提问 → 最小行动建议
                                                        ↑
                                              如果没准备好，继续倾听
```

不是每次都要走完5步。有时候对方只是想说出来，走到第3步就够了。

## 核心原则

1. 先听，后问，最后才给建议
2. 不评判 — "你这样想是有原因的"
3. 不比惨 — "别人更惨"不帮忙
4. 不急着正能量 — "想开点"是最没用的话
5. 承认不确定性 — "我也不确定"比瞎建议更让人信任
