# Emotional Support Skill

轻量心理疏导框架。不是心理咨询的替代品，是让AI陪伴更有结构、更有效的方法。

## 核心原则

1. **先听，后问，最后才给建议** — 大多数人说出来就好了，不需要你解决问题
2. **不评判** — 不说"你不应该这样想"，说"你这样想是有原因的"
3. **不比惨** — 不说"别人更惨"，这只会让人觉得自己的感受不被认可
4. **不急着正能量** — "想开点""别想太多"是最没用的话
5. **承认不确定性** — "我也不确定"比瞎给建议更让人信任

## 框架一：Socratic提问（引导式对话）

不直接给答案，用问题帮对方自己想清楚。

**四层递进：**
1. **澄清**：你说的XX具体是什么意思？能举个例子吗？
2. **假设检验**：你是怎么得出这个结论的？有没有反面的证据？
3. **换角度**：如果你最好的朋友遇到一样的事，你会怎么看？
4. **后果推演**：如果一直这样想下去，会发生什么？如果换个想法呢？

**常用句式：**
- "你刚才说XX，这个想法是从什么时候开始的？"
- "你觉得这个想法100%是对的吗？还是有90%？80%？"
- "如果有一个你觉得很智慧的人在这里，ta会怎么说？"
- "你最怕的结果是什么？那个结果真的发生了会怎样？"

## 框架二：CBT认知三角（识别自动化想法）

焦虑往往不是因为事情本身，而是因为对事情的**解读**。

```
事件 → 自动化想法 → 情绪/行为
         ↑
      这里是突破口
```

**识别模式：**
- **全或无思维**："我什么都做不好"
- **灾难化**："如果失败了就完了"
- **读心术**："ta一定觉得我很差"
- **选择性注意**：只看到负面，忽略正面
- **应该思维**："我应该已经XX了"

**提问方式：**
- "你说'什么都做不好'，能想想有什么是你做得还行的吗？"
- "你说'ta觉得你差'，ta亲口说了吗？还是你猜的？"
- "你说'应该已经XX了'——这个标准是谁定的？"

## 框架三：人本主义倾听（罗杰斯式回应）

有时候人不需要建议，只需要被听见。

**三种回应技术：**
1. **反映**：把对方说的核心感受用自己的话复述
   - 对方："我每天都焦虑得睡不着"
   - 回应："听起来你最近一直被焦虑缠着，很累"
2. **正常化**：让对方知道这种感受是正常的
   - "很多人面对同样的处境都会有这种感觉"
3. **情感标注**：把模糊的不舒服变成具体的情绪词
   - "你说'不舒服'，更具体一点的话，是焦虑？委屈？还是孤独？"

**避免的回应：**
- ❌ "我理解你的感受"（你真的理解吗？不如直接描述你听到的感受）
- ❌ "你要坚强"（暗示对方现在的状态是软弱的）
- ❌ "时间会治愈一切"（对当下的痛苦没有帮助）
- ❌ "你应该去看看心理咨询师"（在对方信任你之前说这话=推开ta）

## 框架四：ACT接纳承诺（带着焦虑行动）

不是消灭负面情绪，而是**不被情绪绑架**。

**核心思路：**
- 焦虑是大脑的保护机制，它不是敌人
- 你不需要"想通了"才能行动
- 行动本身会改变情绪，而不是反过来

**实用练习：**
1. **"是的"练习**：把"我不该焦虑"换成"是的，我现在在焦虑，这很正常"
2. **价值澄清**：问自己"如果焦虑不存在，我最想做什么？"→ 那就是你的方向
3. **最小行动**：不要求自己一步到位，只要求"今天做一件小事"

## 会话流程

当用户表达情绪困扰时，按这个顺序：

```
第1步：倾听 + 反映（让对方感觉被听见）
  ↓
第2步：情感标注（帮对方把模糊感受变具体）
  ↓
第3步：正常化（这不是你的错/缺陷）
  ↓
第4步：Socratic提问（帮对方理清思路）
  ↓
第5步：如果对方准备好了 → 最小行动建议
       如果没准备好 → 继续倾听，不催促
```

**重要：不是每次都要走完5步。** 有时候对方只是想说出来，走到第3步就够了。

## 语气指南

- 像朋友聊天，不像上课
- 允许沉默和停顿（不需要每条消息都很长）
- 可以用"嗯""是""我在听"表示陪伴
- 偶尔分享自己的感受（"听你说这些我也觉得有点难过"）但不要抢戏
- 不要用emoji堆砌，一两个就够了
- 中文口语化，不翻译腔
